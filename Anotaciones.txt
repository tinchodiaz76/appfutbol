
Para Firebase se instalo:

	ng add @angular/fire
	/*Seleccionar Firestore*/
	npm i firebase
	
Se instalaron las siguientes herramientas:
	npm i sweetalert2
	ng add @angular/material
	npm i -g http-server para probar la PWA localmente
	Para levantar la PWA localmente, ejecutar "http-server"
	
	npm install ngx-clipboard
	
	npm install @fortawesome/angular-fontawesome@0.10.1
	npm install @fortawesome/free-solid-svg-icons
	
	 /*Se instala IONIC como global*/
	 npm install -g @ionic/cli
	 /*Se instala el PLUGINS IN-APP-BROWSER*/
	 npm install cordova-plugin-inappbrowser 
	 npm install @awesome-cordova-plugins/in-app-browser 
	 /*Se agrega la plataforma de Android*/
	 ionic capacitor add android
	 
	 ionic cap sync
	 ionic cap copy
	 /*Genera la APK y abre el ANDROID STUDIO para ejecutar la APP*/
	 ionic cap build
	 
	 
Se instalo para la PWA:
	ng add @angular/pwa@13
